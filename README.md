## Running the Code

Clone the repository:

```bash
git clone https://github.com/haithamhroob/binx-ai-ml-internship.git
cd binx-ai-ml-internship
```

Create a virtual environment:

```bash
python3 -m venv .venv
```

Activate it on Linux or Ubuntu:

```bash
source .venv/bin/activate
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run a Python file:

```bash
python3 filename.py
```

To work with Jupyter notebooks:

```bash
jupyter notebook
```

Some folders may include their own README file when additional instructions or explanations are needed.
